一、冒泡排序

依据陶某人的介绍，就是像冒泡一样的排序就叫冒泡排序。
那么什么叫像冒泡一样的排序呢。

简单来说，冒泡排序就是依次比较相邻的元素，如果前者较大，则进行交换。
步骤就是这样：
    1.比较相邻的元素，如果前一个比后一个大，则交换两个元素;
    2.对每一对相邻元素都进行1步骤，从第一对到最后一对。第一轮结束之后，最大的元素则在最后的位置;
    3.进行第二轮比较，应排除上一轮的最后一个元素;
    4.重复上述的步骤，直到没有数字需要交换;

代码实现：

# python
def Bubble_sort(a,n):
    for i in range(n-1):
        for j in range(n-1-i):
            sortble = True   # 用来判断是否已经排好序
            if a[j+1] < a[j]:
                a[j],a[j+1] = a[j+1],a[j]
                sortble = False
        if(sortble):
            break

// C
void Bubble_sort(int arr[],int n)
{
    int i,j,temp;
    bool sortable = false;   // 用来判断是否已经排好序
    for(i=0;i<n;i++)
    {
        for(j=0;j<n-i-1;j++)
        {
            if (arr[j]>arr[j+1])
            {
                temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
                sortable = true;
            }
        }
        if(!sortable){
            break;
        }
    }
}

二、快速排序

简单来说，快速排序是冒泡排序的升级版。
步骤就是这样：
    1.设置两个变量L、R，分别为数组的第一个位置和最后一个位置，第一轮一般为0和n-1，即L=0，R =n-1；
    2.以第一个元素为关键数据key，即key = a[L]；
    3.从后往前搜索，找到第一个小于key的值a[R]，交换A[L]、A[R]；
    4.从前往后搜索，找到第一个大于key的值a[L]，交换A[L]、A[R]；
    5.重复3，4步骤，直到L=R，此时L左边的值均小于key，L右边的值均大于key；
    6.将数组分为两部分，再采用递归继续进行快速排序，最终得到有序序列。

代码实现：

# python 分治+递归
def Quick_sort(a):
    if len(a)<2:
        return a
    else:
        key = a[0]
        less = [l for l in a if l < key]
        equal = [e for e in a if e == key]
        greater = [g for g in a if g > key]
        return Quick_sort(less) + equal + Quick_sort(greater)    

# C 递归
void Quick_sort(int arr[], int Lindex, int Rindex)
{
    int L,R,key;
    L = Lindex;
    R = Rindex;
    key = arr[L];
    while(L<R)
    {
        while(L<R && arr[R]>=key) R--;
        arr[L] = arr[R];
        while(L<R && arr[L]<=key) L++;
        arr[R] = arr[L];
    }
    arr[L] = key;
    if(L-Lindex>1)
    {
        Quick_sort(arr,Lindex,L-1);
    }
    if(Rindex-R>1)
    {
        Quick_sort(arr,R+1,Rindex);
    }
}

三、插入排序

简单来说，插入排序就是将未排序的元素插入到有序序列中。
步骤就是这样：
    1.从有序序列和无序序列{a2,a3，…，an}开始进行排序；
    2.处理第i个元素时（i=2,3，…，n），序列{a1,a2，…，ai-1}是已有序的，而序列{ai,ai+1，…，an}是无序的。用ai与ai-1，a i-2，…，a1进行比较，找出合适的位置将ai插入；
    3.重复第二步，共进行n-1次插入处理,序列全部有序。

代码实现：

# python
def Insert_sort(a,n):
    for i in range(1,n):
        for j in range (i,0,-1):
            if a[j]<a[j-1]:
                a[j],a[j-1] = a[j-1],a[j]

// C
void Insert_sort(int arr[],int n)
{
    int i,j,temp;
    for(i=1;i<n;i++)
    {
        if(arr[i]<arr[i-1])
        {
            temp = arr[i];
            for(j=i;j>0;j--)
            {
                if(temp<arr[j-1])
                {
                    arr[j] = arr[j-1];
                }
                else break;
            }
            arr[j] = temp;
        }
    }
}

四、希尔排序

简单来说，就是插入排序的升级版。
步骤就是这样：
    1.设置增量d,初始值一般为n/2，将序列进行分组；
    2.对于每一组序列进行插入排序；
    3.然后缩小增量，继续进行分组插入排序；
    4.直到d<1,序列全部有序。

代码实现：

# python
def Shell_sort(a,n):
    d = int(n/2)
    while d>=1:
        for i in range(d):
            for j in range (i,n,d):
                for k in range (j,n,d):
                    if a[j]>a[k]:
                        a[j],a[k] = a[k],a[j]
        d = int(d/2)

// C
void Shell_sort(int arr[],int n)
{
    int d,i,j,k,temp;
    d = n/2;
    while(d>=1)
    {
        for(i=0;i<d;i++)
        {
            for(j=i;j<n;j+=d)
            {
                for(k=j;k<n;k+=d)
                {
                    if(arr[k]<arr[j])
                    {
                        temp = arr[k];
                        arr[k] = arr[j];
                        arr[j] = temp;
                    }
                }
            }
        } 
        d = d/2;   
    }
}

五、选择排序

简单来说，就是选出无序序列中的最小值，放入相应的位置。
步骤就是这样：
    1.初始状态：序列全部无序；
    2.第一趟排序，在无序序列中找出最小的值，然后与第一个值交换，此时第一个有序，后面n-1个值无序；
    3.第i趟排序，前i-1个值为有序序列，后n-i-1个值无序，在无序序列中找出最小值，与第i个值交换；
    4.直到第n趟排序，得到有序序列。

代码实现：

# python
def Select_sort(a,n):
    for i in range(n):
        min = i
        for j in range (i,n):
            if a[j]<a[min]:
                min = j
        a[i],a[min] = a[min],a[i]

// C
void Select_sort(int arr[],int n)
{
    int i,j,temp,min;
    for(i=0;i<n;i++)
    {
        min = i;
        for(j=i;j<n;j++)
        {
            if (arr[j]<arr[min])
            {
                min = j;
            }

        }
        if(min!=i)
        {
            temp = arr[i];
            arr[i] = arr[min];
            arr[min] = temp;
        }
    }
}

六、堆排序

简单来说，就是构造大顶堆，然后交换数值。然而大顶堆就是类似于二叉树，所有父节点均大于其子节点。
步骤就是这样：
    1.首先将n个数值构造成大顶堆；
    2.将堆顶数值与堆底数值交换，然后将第一个到第n-i个数值重新调整为大顶堆；
    3.重复步骤2，得到有序序列。

代码实现：

# python
def Build_MaxHeap(a,start,end):
    dad = start
    son = dad*2+1
    while(son<end):
        if(son+1<end and a[son]<a[son+1]):
            son+=1
        if(a[dad]<a[son]):
            a[dad],a[son] = a[son],a[dad]
            dad = son
            son = dad*2+1
        else: break

def Heap_sort(a,n):
    for i in range (int(n/2)-1,-1,-1):
        Build_MaxHeap(a,i,n)
    for i in range(n-1, 0,-1):
        print(a)
        a[i],a[0] = a[0],a[i]
        Build_MaxHeap(a,0,i)


// C
void Build_MaxHeap(int arr[],int start,int end)
{
    int dad,son,i,j,temp;
    dad = start;
    son = dad*2+1;
    while(son<end)
    {
        if(son+1<end && arr[son]<arr[son+1])  son++;
        if(arr[dad]<arr[son])
        {
            temp = arr[dad];
            arr[dad] = arr[son];
            arr[son] = temp;
            dad = son;
            son = dad*2+1;
        }
        else break;
    }
}

void Heap_sort(int arr[],int n)
{
    int i,j,temp;
    for(i=n/2-1;i>=0;i--)
    {
        Build_MaxHeap(arr,i,n);
    }
    for(j=n-1;j>0;j--)
    {
        temp = arr[j];
        arr[j] = arr[0];
        arr[0] = temp;
        Build_MaxHeap(arr,0,j);
    }
}

七、合并排序（也叫归并排序）

简单来说，就是将两个有序的序列合并成一个有序序列。
步骤就是这样：
    1.先将序列一分为二，利用递归，将子序列分最小即只有一个元素，则此时左序列只有一个元素，右序列为1-2个元素（2个需要继续分，然后进行合并排序）；
    2.左序列与右序列合并，此时的左序列为有已排序三个元素，此时的右序列也应如左序列有3-4个元素，且已排序；
    3.然后依次合并序列，得到有序序列。

代码实现：

# python
def Merge(left,right):
    i,j = 0,0
    result = []
    while(i<len(left) and j<len(right)):
        if left[i]<right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result += list(left[i:])
    result += list(right[j:])
    return result

def Merge_sort(a):
    if len(a)<=1:
        return a
    num = int(len(a)/2)
    left = Merge_sort(a[:num])
    right = Merge_sort(a[num:])
    return Merge(left,right)



// C
void Merge(int arr[],int start,int mid,int end)
{
    int x,y,i,j,l,r;
    x = mid - start + 1;
    y = end - mid;
    int *L = new int[x];
    int *R = new int[y];
    for(i=0;i<x;i++)
    {
        L[i] = arr[start+i];
    }
    for(i=0;i<y;i++)
    {
        R[i] = arr[mid+i+1];
    }
    l = 0; r = 0;j=0;
    while(l<x && r<y)
    {
        if (L[l]<R[r]) arr[start+j] = L[l++];
        else   arr[start+j] = R[r++];
        j++;
    }
    while(l<x)
    {
        arr[start+j] = L[l++];
        j++;
    }
    while(r<y)
    {
        arr[start+j] = R[r++];
        j++;
    }
    delete L,R;
}

void Merge_sort(int arr[],int start,int end)
{
    int mid;
    if(start<end)
    {
        mid = (start+end)/2;
        Merge_sort(arr,start,mid);
        Merge_sort(arr,mid+1,end);
        Merge(arr,start,mid,end);
    }
}

八、计数排序（也叫归并排序）

简单来说，就是若小于x的元素有n个，则说明第n+1个数是x，即 a[n] = x
步骤就是这样：
    1.首先会对每个输入进行频率统计，得到元素的频率表；
    2.然后将频率表转换为该元素的开始索引；
    3.据各个元素的开始索引，将相同元素分类到临时数组中；
    4.最后将临时数组中的元素写回到原数组中。

代码实现：

# python
def Count_sort(a,n):
    b = [0 for i in range(n)]
    for i in range(n):
        small = 0
        equal = 0
        for j in range(n):
            if a[j]<a[i]:
                small += 1
            elif a[j] == a[i]:
                equal += 1
        for k in range (small,small+equal):
            b[k] = a[i]
    a[:] = b[:]

// C
void Count_sort(int arr[],int n)
{
    int max = arr[0];
    int i,j,k;
    for(i=0;i<n;i++)
    {
        if(arr[i]>max)  max = arr[i];
    }
    max++;
    int *b = new int[max]{0};
    int *c = new int[n]{0};
    for(i=0;i<n;i++)
    {
        b[arr[i]]++;
    }
    for(i=1;i<max;i++)
    {
        b[i] += b[i-1];
    }
    for(i=0;i<n;i++)
    {
        c[b[arr[i]]-1] = arr[i];
        b[arr[i]]--;
    }
    for(i=0;i<n;i++)
    {
        arr[i] = c[i]; 
    }
    delete b,c;
}

九、桶排序

简单来说，就是分出N个桶，然后把序列中的元素丢到桶里，进行桶内快排，再将桶连起来，其实与计数有点类似。
步骤就是这样：
    1.分N个桶，每个桶的间隔为M，因此N*M>max；
    2.将序列中的每个元素分别的放入桶中；
    3.桶内进行快速排序；
    4.再把每个桶连起来，得到有序序列。

代码实现：

# python
def Bucket_sort(a,n):
    max,min = a[0],a[0]
    for i in range(1,n):
        if a[i]>max :
            max = a[i]
        if a[i]<min:
            min = a[i]
    gap = int((max-min)/n + 1)
    bucket_num =int((max-min)/gap + 1)
    bucket = [[]for i in range(bucket_num)]
    for i in range(n):
        bucket[int(a[i]/gap)].append(a[i])
    result = []
    for i in range(bucket_num):
        result += Quick_sort(bucket[i])
    return result

十、基数排序

简单来说，就是按照每一位进行排序,其实和桶排序类似，各个位的数丢到对应的桶中。
步骤就是这样：
    1.得到序列最大值的位数digit；
    2.分10个桶，分别表示当前位数值为0-9；
    3.第一轮排个位，即个位为1则放入1桶中，个位为3则放入3桶中……再将10个桶连起来；
    4.进行第二轮排十位……
    5.第digit轮排完，得到有序序列。

代码实现：

# python
def Radix_sort(a,n):
    max = a[0]
    for i in a:
        if i>max:
            max =i
    digit = len(str(max))
    div = 1
    for i in range(digit):
        bucket = [[] for i in range(10)]
        for j in a:
            bucket[int(j/div)%10].append(j)
        result = []
        for j in range(10):
            result += bucket[j]
        a[:] = result[:]
        div *= 10

// C
void Radix_sort(int arr[], int n)
{
    int digit,i,p,j,k,radix;
    p = 1;digit = 0;radix = 1;
    for(i=0;i<n;i++)
    {
        if(arr[i]>=p)
        {
            p = p*10;
            digit++;
        }
    }
    for(i=0;i<digit;i++)
    {
        int *b = new int[10]{0};
        int *c = new int [n];
        for(j=0;j<n;j++)
        {
            k = (arr[j]/radix) % 10;
            b[k]++;
        }
        for(j=1;j<10;j++)
        {
            b[j] += b[j-1];
        }
        for(j=0;j<n;j++)
        {
            k = (arr[j]/radix) % 10;
            c[b[k]-1] = arr[j];
            b[k]--;
        }
        for(j=0;j<n;j++)
        {
            arr[j] = c[j];
        }
        radix *= 10;
        delete b,j;
    }
}
