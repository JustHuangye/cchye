python的部分常用操作(持续更新)

实习的过程中，某徐姓组长觉得我学有余力，遂安排了一波用python进行工具开发，从此我成为三人实习小队中的最大赢家。
由于python也是初次接触，基本语法方面网上有很多教程，比如菜鸟驿站、廖雪峰等，下面就简单的介绍一些比较常用的一些操作。

正则表达式

  1. 模块依赖
    import re
  2. 简单运用
    re.findall('x(.*)y',string)   # 获得xy中间的数据,返回列表
# 获取列表长度len 获取np数组大小shape

对文件的操作

  1. 打开文件
    x = open('filename','r/w/a...')  # r只读，w写，a多次写
  2. 读取文件中的内容
    x.read()  # 返回字符串
  3. 向文件中写入
    x.write('string')
  4. 关闭文件
    x.close()

numpy的使用

  1. 引用模块
    import numpy as np
  2. 创建用0填充的数组
    np.zero((row,col,channel),dtype = np.int8) # shape分别为数组的行，列，通道数；dtype为数据类型
# 返回一个给定形状和类型的用0填充的数组
  3. 从文件中读取数据
    np.fromfile('filename',dtype = np.int8) #从文件中按指定类型读取数据

检查并得到字符串中的数字

  1. 简单运用函数

     num = []
 for i in string:
     if i.isdigit():    #检测字符串是否只由数字组成。
         num.append(i)

    同样，isalpha() 检测字符串是否只由字母组成, isalnum()检测字符串是否由字母和数字组成……

  2. 正则表达式

     s = re.findall("\d+",string)

  3. filter
    filter() 函数用于过滤序列，过滤掉不符合条件的元素，返回由符合条件元素组成的新列表。

     num = filter(str.isdigit, string)  #注意: Pyhton2.7 返回列表，Python3.x 返回迭代器对象
 result = list(num) #python3.x 将迭代器对象转换为列表
